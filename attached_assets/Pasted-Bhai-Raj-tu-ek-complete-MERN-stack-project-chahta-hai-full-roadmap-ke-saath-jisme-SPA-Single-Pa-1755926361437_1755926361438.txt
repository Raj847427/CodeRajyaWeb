👑 Project Name: CodeRajya
Tagline: “Jahan har coder banega king.”
Type: Full-stack MERN SPA (Single Page Application)
Goal: Skill-based learning + mentorship + interview prep + community support

🧭 Phase-Wise Breakdown
🔹 Phase 1: Planning & Setup
- ✅ Finalize modules: Skill Learning, Mentor Connect, Interview Prep, Doubt Forum, Progress Tracker
- ✅ Choose tech stack:
- Frontend: React + React Router DOM (SPA)
- Backend: Node.js + Express
- Database: MongoDB
- Auth: JWT + bcrypt
- Styling: TailwindCSS or Material UI
- State Management: Context API or Redux (for scalability)

🔹 Phase 2: Folder Structure
CodeRajya/
├── client/              # React SPA
│   ├── src/
│   │   ├── components/  # Reusable UI
│   │   ├── pages/       # Home, Dashboard, Modules
│   │   ├── context/     # Auth & global state
│   │   ├── utils/       # API calls, helpers
│   │   └── App.js       # Routes setup
├── server/              # Node + Express
│   ├── routes/          # API endpoints
│   ├── controllers/     # Logic handlers
│   ├── models/          # MongoDB schemas
│   ├── middleware/      # Auth, error handling
│   └── server.js        # Entry point

🔹 Phase 3: Core Modules
🧠 1. Skill Modules
- Topics: Java, DSA, MERN, OOP, etc.
- Each module has:
- Lessons (text + video)
- Quizzes
- Code challenges
- Backend tracks progress per user
🧑‍🏫 2. Mentor Connect
- Mentor profiles with skill tags
- Booking system (date/time slots)
- Chat system (Socket.io or thread-based)
- Optional: Video call integration (Jitsi or Zoom API)
🎯 3. Interview Prep Zone
- Real questions with explanations
- Code editor (Monaco Editor)
- Test case validation
- Leaderboard for top performers
❓ 4. Doubt Forum
- Post doubts with tags
- Community answers + mentor replies
- Upvote/downvote system
- Searchable archive
📊 5. Progress Tracker
- Skill graph per user
- Badges for milestones
- Weekly email summary (optional)
🧑‍💼 6. Admin Panel
- Add/edit modules
- Approve mentors
- View analytics (active users, top learners)
